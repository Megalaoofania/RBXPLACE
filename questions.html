Why does my antivirus say that RBXIDLE is a virus?

Cryptocurrency miners (like RBXIDLE) suffer this same issue because the average user will not be mining cryptocurrency, and this can also be used for malicous intent if the user is unaware.


Are any other devices supported?

No. RBXIDLE is only built for Windows platforms, and is not expected to arrive on other devices anytime soon.


What does it mean when RBXIDLE says "Unsupported"?

Unfortuantely, if it says this your computer will not be able to run RBXIDLE.


Can you CPU mine?

Effective June 1st, 2023, CPU mining support was dropped because CPU mining was not very profitable, and could potentially fry your CPU.
